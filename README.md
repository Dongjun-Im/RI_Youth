# 열린관광지 모니터링 조사 웹앱

참가자가 **휴대폰번호로 로그인**하여 「열린관광지 모니터링 조사지」를 개인별로 작성·제출하는 웹앱.
참가자가 5개 대분류 탭의 모든 필수 항목을 완료하면 **제출**할 수 있고, **등록된 전원이 제출을 마치면
운영진에게 자동으로 완료 알림 메일**이 발송됩니다.

## 주요 기능
- 📱 등록된 휴대폰번호로 간편 로그인 (비밀번호 없음)
- 🗂️ **5개 대분류 탭**: 설문조사 / 이동권 / 시설편의 / 정보접근권 / 문화향유권
- 🧩 **3가지 문항 유형**
  - **선택(라디오)** — 5점 척도(매우 그렇지 않다~매우 그렇다), 지표(적합/애매/미흡 등). 선택지는 항목마다 다르게 설정
  - **서술(텍스트)** — 기타 의견 등 자유 입력(선택)
  - 적합/애매/미흡 지표에는 **개선 사항** 서술란(선택) 제공
- 💾 **개인별 응답 + 자동 저장** (조 단위 공유 아님)
- ✅ 모든 필수 항목 완료 시 **제출** → 제출 후 **잠금**(수정 불가). 제출 시 **조사원 구분·관광지명** 입력
- 🔊 **스크린리더 지원**: 남은 항목 수를 `aria-live` 영역으로 실시간 안내
- 📧 **전원 제출 완료 시 자동 메일 1회** (중복 발송 방지) + 관리자 **강제 발송** 버튼
- 🖥️ 관리자: 참가자·조 관리(개별/CSV), **체크리스트(탭·항목·유형·선택지) 관리**, 제출 현황·순위, 응답 CSV 내보내기

## 설치 (Windows / XAMPP 기준)

### 1. XAMPP 설치 및 실행
- https://www.apachefriends.org 에서 설치 후 **Apache**·**MySQL** 시작

### 2. 프로젝트 배치
`C:\xampp\htdocs\gagopa\` 에 파일을 복사

### 3. 데이터베이스 생성 + 체크리스트 시딩
```
# 1) 테이블 + 샘플 조/참가자 생성
C:\xampp\mysql\bin\mysql.exe -u root < schema.sql

# 2) 5개 탭 + 조사지 전체 항목(72개) 시딩
C:\xampp\php\php.exe seed_checklist.php
```
(항목은 관리자 → **체크리스트 관리** 에서 언제든 추가/수정/삭제 가능)

### 4. 설정 — `config.php`
`APP_TITLE`, `DB_*`, `ADMIN_PASSWORD`, `NOTIFY_TO`(완료 알림 수신), `SMTP_*`(Gmail 앱 비밀번호) 를 실제 값으로.
- 메일을 실제 발송하지 않고 로그만 남기려면 `MAIL_DRY_RUN = true` (→ `mail_dryrun.log`)

### 5. 접속
- 참가자: http://localhost/gagopa/
- 관리자: http://localhost/gagopa/admin.php

## 사용 흐름
1. **관리자**가 참가자(조·이름·휴대폰)를 등록 (개별 또는 CSV 일괄)
2. **참가자**가 휴대폰번호로 로그인 → 5개 탭의 문항에 응답(자동 저장)
3. 모든 필수 항목 완료 + 조사원 구분·관광지명 입력 → **제출**(이후 잠금)
4. **등록된 전원 제출** 시 운영진에게 완료 메일 자동 발송 (미제출자가 있으면 관리자가 강제 발송 가능)

## 🧪 테스트

### 자동 테스트 (MySQL 불필요, 인메모리 SQLite)
```
C:\xampp\php\php.exe tests\run_tests.php     또는  tests\run.bat
```
로그인·응답 저장·진행률·제출/잠금·전원 제출 메일·중복 방지·항목 관리·CSV 등 25개 검증.

### 수동 테스트
- 참가자 화면에서 응답 → 새로고침 시 유지, 남은 항목 수 감소 확인
- 모바일: 크롬 F12 → 기기 툴바(Ctrl+Shift+M)
- 스크린리더(NVDA 등)로 "남은 항목 N개" 안내 확인
- 관리자 대시보드에서 제출 현황·순위·CSV·강제 발송 확인

## 파일 구조
```
gagopa/
├─ config.php          # 설정 (define 가드 — 테스트에서 오버라이드 가능)
├─ db.php              # PDO 연결(주입 가능) + INSERT IGNORE 호환 헬퍼
├─ schema.sql          # 테이블 + 샘플 조/참가자
├─ seed_checklist.php  # 조사지 전체(5탭·72항목) 시딩
├─ lib.php             # 인증·응답·진행률·제출·완료메일·관리 로직
├─ smtp.php            # 순수 PHP SMTP 발송
├─ index.php / logout.php
├─ app.php             # 참가자: 탭 + 문항(라디오/텍스트) + 제출
├─ api/save.php        # 응답 자동 저장
├─ api/submit.php      # 제출(잠금)
├─ admin.php           # 관리자 컨트롤러(로그인·라우팅·CSV·강제발송)
├─ admin_dashboard.php # 제출 현황·순위
├─ admin_members.php   # 참가자·조 관리(+CSV 일괄등록)
├─ admin_checklist.php # 탭·항목(유형·선택지·개선사항·필수) 관리
├─ assets/ (style.css, app.js)
└─ tests/ (bootstrap.php, run_tests.php, run.bat)
```
